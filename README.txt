# GraphicPrimitivesLibrary
Simple set of graphic primitives, line, ellipse, etc. based on SDL. Code for drawing text is early. Moderate efficency, no apparent bugs, 32BPP only.

Requirements:

SDL library, version 1.2

Usage:

To use it, simply include graphic.cc file to your C++ code, defining symbol ALE_ZLY_STYL_BIBLIOTEK before.

To display rendered graphic, one needs to call Projektor::odswiez(); at end of rendering code.
